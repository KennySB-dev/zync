About Zync

Zync is one of the best multi-functional bot on discord. It is completely free, includes a bunch of commands, has a very fast support system, and has an uptime of 99.9%

Technical

Zync is built with Javascript using the discord.js module. The bot uses it's own self hosted database to to store your info in a secure, and safe way. With the help of our super fast and flexible database we are able to provide you the fastest response time possible.

Getting Started

To get started with Zync is a fairly easy process as I have done my best to make it simpler for you and other members.

In-bot help

There is also an in-bot getting started command that you can use to help you get started.
You also have the option to get help on all the bot commands right through discord with the use of the bot's internal help command.

Moderation

  • Announce
    • Description : Announce a message in a certain channel!
    Usage : /announce
    Note : You cannot use headers, so #. ## or ### in the embed.
  • Transcript
    • Description : Get a transcript of the channel the command is ran in.
    Usage : /transcript
    Note : Run the command again if it doesn't load the first time.
  • Snipe
    • Description : Get a recently deleted message.
    Usage : /snipe
    Note : Run the command again if it doesn't load the first time.
  • Un/Ban
    • Description : Ban a user.
    Usage : /ban add [reason]
    Description : Unban a user.
    Usage : /ban remove
  • Kick
    • Description : Kick a user!
    Usage : /kick
  • Timeout
    • Description : Timeout a member!
    Usage : /timeout
    Note : The duration HAS to be in minutes, for a good time to minutes calculator dh < command name>
    You can do dh help < command name> to get more info about a command.
